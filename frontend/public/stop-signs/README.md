# Таблички на зупинки з QR-кодом

## Файли

- **stop-sign-11-prozhektor.html** — редагуваний шаблон (відкрий у браузері або редакторі)
- **stop-sign-11-prozhektor.pdf** — готовий PDF для друку

## Редагування

1. Відкрий `.html` у будь-якому текстовому редакторі
2. Зміни текст (маршрут, зупинка, напрямок)
3. Для іншого напрямку зміни `dir=back` на `dir=there` у `src` зображення QR (або навпаки)
4. Для іншого маршруту/зупинки — зміни URL у `data=` параметрі QR

## Генерація PDF

**Варіант 1 — з браузера:**
1. Відкрий `.html` у Chrome
2. Ctrl+P (або Cmd+P) → «Зберегти як PDF»

**Варіант 2 — скрипт:**
```bash
cd scripts && npm install
node generate-stop-sign-pdf.js
```

PDF збережеться поруч з HTML.

## Формат посилання для QR

```
https://malin.kiev.ua/localtransport/route/{routeId}?stop={stopName}&dir={there|back}
```

- `dir=there` — напрямок до кінцевої «to» (напр. на Вокзал для маршруту 11)
- `dir=back` — напрямок до кінцевої «from» (напр. на Паперову фабрику для маршруту 11)
