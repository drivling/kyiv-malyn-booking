# UX Jakdojade — що відтворено та що ні

## Відтворено максимально близько

### Головна сторінка (`/localtransport`, `/localtransport/З/До`)
- **Поля «З» та «До»** — два основні поля з автодоповненням, кнопка ⇄ для обміну.
- **Дата і час** — формат ДД.ММ.РР та `input type="time"`, як у Jakdojade (параметри `d`, `h`).
- **Кнопка «Знайти»** — акцентна, в один ряд з датою/часом.
- **URL** — формат `/localtransport/FromStop/ToStop?d=DD.MM.YY&h=HH:mm` як у Jakdojade.
- **Список маршрутів** — тільки маршрути, що з’єднують обрані зупинки; нумерація лінії, кінцеві, кількість рейсів.
- **Тема Jakdojade (wtk-red)** — червоний акцент `#E30613`, сірі фони, системний шрифт.

### Сторінка результату (`/localtransport/route/:id?stop=...&to=...&time=...&dir=...`)
- **Верхній бар** — кнопка «назад», номер лінії (пілюля), текст «З → До», дата та час справа.
- **Заголовок маршруту** — № лінії + напрямок, кнопки перемикання напрямку (→ кінцева / ← початкова).
- **Секція «Розклад руху»** — заголовок, поля З/До з ⇄, кнопка «Друк».
- **Таблиця відправлень (tablica)** — темний заголовок (у темі Jakdojade — червоний), великі часи (1.25rem), рядки «Відправлення | Прибуття», підсвітка обраного рейсу.
- **Секція «Зупинки на маршруті»** — заголовок, карта + список зупинок з часом прибуття та тривалістю до наступної; виділення «З» (червоний) та «До» (синій у темі Jakdojade).

## Що не вдалося відтворити 1:1 (і чому)

| Елемент | Причина |
|--------|--------|
| **Точні кольори hex** | У документації Jakdojade (widgets, docs) вказані лише назви тем `wtk-red` та `jd-purple`, без hex. Використано типовий транспортний червоний `#E30613`. |
| **Піксельні відступи/розміри** | Сторінка jakdojade.pl при fetch дає таймаут; немає живого референсу для вимірювань. Відступи/розміри підібрані «на око» під опис з company.jakdojade.pl. |
| **Міні-карта в верхньому барі** | У Jakdojade в результаті часто є міні-карта або прев’ю маршруту вгорі. Ми не додавали окремий міні-компонент карти в бар, щоб не ускладнювати layout; карта лишається в блоці «Зупинки на маршруті». |
| **Поведінка «прибуття о» (ia=true)** | У Jakdojade є режим «прибуття о заданому часі». У нас пошук лише «відправлення о»; логіка зворотного пошуку за часом прибуття не реалізована. |
| **Інші мови / локалі** | Jakdojade підтримує зміну мови (наприклад `locale=pl`). Наша сторінка тільки українською. |
| **Віджети та зовнішня вбудовка** | У нас немає аналога Planner Widget Jakdojade для вбудовки в інші сайти. |

Якщо з’явиться скрін або жива сторінка Jakdojade, можна підлаштувати відступи, розміри та (за бажанням) додати міні-карту в бар і режим «прибуття о».
